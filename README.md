Genre Match is a Python desktop application that helps users discover which music genres best match their preferences. Users answer a series of questions about their musical tastes, and the application calculates their top genre matches based on their responses.

Features:

Interactive music personality quiz

15 genre-based questions

Supports 10 music genres

Displays Top 3 genre matches

Genre descriptions included

Simple Tkinter graphical interface

Restart quiz functionality

Saves results to a text file
